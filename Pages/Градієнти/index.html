Градієнти

linear-gradient

Лінійний градієнт відтворює перехід кольорів вздовж лінійного напрямку.

linear-gradient(blue, yellow) створює градієнт, який починається з синього кольору та закінчується жовтим.

HTML
CSS

HTML


<div class="divGrad" style="background: linear-gradient(blue, yellow);"></div>

CSS

Напрямок

Градієнти можуть мати різні напрямки, вказані за допомогою кута або ключових слів, таких як to top, to bottom, to left, to right, to top left, to top right, to bottom left, to bottom right.

linear-gradient(to left, blue, yellow) створить градієнт, який йде вліво від синього до жовтого.

HTML
CSS

HTML


<div class="divGrad" style="background: linear-gradient(to left, blue, yellow);"></div>
<div class="divGrad" style="background: linear-gradient(to right bottom, blue, yellow);"></div>
<div class="divGrad" style="background: linear-gradient(45deg, blue, yellow);"></div>

CSS

Відсотки

Вказання відсотків дозволяє контролювати місцезнаходження кольорів в градієнті.


Наприклад: linear-gradient(blue 30%, yellow) створить градієнт, який почнеться з синього на 30% ширини, а потім перейде до жовтого.

HTML
CSS

HTML


<div class="divGrad" style="background: linear-gradient(blue 30%, yellow);"></div>
<div class="divGrad" style="background: linear-gradient(blue 50%, yellow 50%); width: 450px;"></div>
<div class="divGrad" style="background: linear-gradient(green 25%, yellow 25% 50%, red 50% 75%, blue 75%); width: 450px;"></div>

CSS

radial-gradient

Радіальний градієнт відтворює перехід кольорів від одного центру до краю.

Наприклад: radial-gradient(blue, yellow) створює градієнт, який починається з синього кольору в центрі та закінчується жовтим на краю.

HTML
CSS

HTML


<div class="divGrad" style="background: radial-gradient(blue , yellow);"></div>
<div class="divGrad" style="background: radial-gradient(blue , yellow 30%);"></div>

CSS

circle at

Можна вказати радіус кута для радіальних градієнтів, щоб визначити, звідки починається центр градієнту.

Наприклад: radial-gradient(circle at 75px 75px, blue, yellow) створить радіальний градієнт з центром в точці (75px, 75px).

HTML
CSS

HTML


<div class="divGrad" style="background: radial-gradient(circle at 75px 75px, blue, yellow 30%);"></div>
<div class="divGrad" style="background: radial-gradient(circle at 0px 150px, blue , yellow);"></div>
<div class="divGrad" style="background: radial-gradient(circle at 75px 75px, blue 70%, yellow 30%); border-radius: 25%;"></div>

CSS

conic-gradient

Конічний градієнт створює перехід кольорів вздовж кільця.

Наприклад: conic-gradient(blue, yellow, green, red) створює градієнт, який починається з синього, йде до жовтого, потім до зеленого і закінчується червоним.

HTML
CSS

HTML


<div class="divGrad" style="background: conic-gradient(blue, yellow, green, red);"></div>
<div class="divGrad" style="background: conic-gradient(from 45deg, blue, yellow, green, red);"></div>
<div class="divGrad" style="background: conic-gradient(at 225px 225px, blue, yellow, green, red);"></div>

CSS

repeating

Вони використовуються для створення градієнтів, які повторюються через певний інтервал.

Наприклад: repeating-linear-gradient(blue 10%, yellow 20%) створює градієнт, який починається з синього на 10% ширини, потім жовтий на 20%, і так далі.

HTML
CSS

HTML


<div class="divGrad" style="background: repeating-linear-gradient(blue 10%, yellow 20%);"></div>
<div class="divGrad" style="background: repeating-linear-gradient(blue 0% 10%, yellow 10% 20%);"></div>
<div class="divGrad" style="background: repeating-radial-gradient(blue 0% 10%, yellow 10% 30%);"></div>
<div class="divGrad" style="background: repeating-conic-gradient(blue 10%, yellow 20%);"></div>

CSS
