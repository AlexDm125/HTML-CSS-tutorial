Тег <iframe>

Тег <iframe> використовується для вставки іншого HTML-документа в поточний документ. Це часто використовується для вбудовування відео з YouTube, карт з Google Maps та інших ресурсів.

Приклад

Атрибути

  • width: Вказує ширину iframe в пікселях. У цьому прикладі - 560.
  • height: Вказує висоту iframe в пікселях. У цьому прикладі - 315.
  • src: Вказує URL вбудованого контенту. У цьому прикладі - YouTube відео.
  • title: Текстовий опис для iframe. У цьому прикладі - "YouTube video player".
  • frameborder: Визначає наявність рамки навколо iframe. Значення "0" означає, що рамки немає.
  • allow: Визначає дозволені дії для вмісту в iframe, такі як використання акселерометра, автозапуск, доступ до буфера обміну, зашифровані медіа, гіроскоп, функція "картина в картині" та веб-поширення.
  • referrerpolicy: Встановлює політику пересилання заголовка Referer. У цьому прикладі - "strict-origin-when-cross-origin".
  • allowfullscreen: Дозволяє повноекранний режим для відео.

HTML


<iframe width="560" height="315" src="https://www.youtube.com/embed/aRGdDy18qfY?si=PahlRTDlzEGMeCcW" title="YouTube video player" frameborder="0" 
allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" 
referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

Тег <audio>

Тег <audio> використовується для відтворення звукових файлів на веб-сторінці.

Приклад

Атрибути для <audio>

  • controls: Додає елементи керування відтворенням (відтворити/пауза, регулювання гучності тощо).

Атрибути для <source>

  • src: Вказує шлях до аудіофайлу. У цьому прикладі - "audio/audio.mp3".
  • type: Вказує тип аудіофайлу. У цьому прикладі - "audio/mpeg".
HTML
CSS

HTML


<audio controls>
    <source src="audio/audio.mp3" type="audio/mpeg">
</audio>

Тег <video>

Тег <video> використовується для відтворення відеофайлів на веб-сторінці.

Атрибути для <video>

  • controls: Додає елементи керування відтворенням (відтворити/пауза, регулювання гучності, перемотування тощо).
  • width: Вказує ширину відеоплеєра в пікселях. У цьому прикладі - 400.

Атрибути для <source>

  • src: Вказує шлях до відеофайлу. У цьому прикладі - "video/2278095-hd_1920_1080_30fps.mp4".
  • type: Вказує тип відеофайлу. У цьому прикладі - "video/mp4".
HTML
CSS

HTML

 
<video controls width="400">
    <source src="video/2278095-hd_1920_1080_30fps.mp4" type="video/mp4">
</video>
