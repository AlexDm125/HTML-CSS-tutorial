Форми

HTML-форми є основним способом збору даних від користувачів на веб-сторінках.

Тег <form>

Тег <form> визначає форму для збору користувацьких даних.

<form action="" name="form1"> </form>

Атрибути

  • action: Визначає URL-адресу, куди буде відправлено дані форми після їх подання.
  • name: Вказує ім'я форми.

Тег <label>

Тег <label> визначає підпис для елементу введення, що покращує доступність форми.

<label for="email">Enter your email</label>

Атрибути

  • for: Зв'язує підпис з конкретним елементом введення через його атрибут id. У деяких випадках значення порожнє.

Тег <input>

Тег <input> визначає поле введення даних.

Типи та атрибути

type="text": Текстове поле введення.


HTML
CSS

HTML

 
<form action="" name="form1">
    <input type="text" name="email" size="20" maxlength="20" value="email"><br>
</form>

CSS
Атрибути
  • name: Ім'я поля введення. У цьому прикладі - "email".
  • size: Ширина поля введення (кількість символів). У цьому прикладі - 20.
  • maxlength: Максимальна кількість символів, які можна ввести. У цьому прикладі - 20.
  • value: Початкове значення поля. У цьому прикладі - "email".

type="checkbox": Прапорець (чекбокс).


HTML
CSS

HTML

 
<form action="" name="form1">
    <label for="" id="checkbox1">
        <input type="checkbox" name="color1" id="red" value="red">Red <br>
        <input type="checkbox" name="color2" id="orange" value="orange">Orange <br>
        <input type="checkbox" name="color3" id="green" value="green">Green <br>
        <input type="checkbox" name="color4" id="blue" value="blue">Blue <br>
        <input type="checkbox" name="color5" id="gold" value="gold">Gold <br>
    </label>
</form>

CSS
Атрибути
  • name: Ім'я прапорця. У прикладі є кілька чекбоксів з іменами "color1", "color2", "color3", "color4", "color5".
  • id: Унікальний ідентифікатор прапорця. У прикладі - "red", "orange", "green", "blue", "gold".
  • value: Значення, яке передається при виборі прапорця.

type="radio": Перемикач (радіо-кнопка).


HTML
CSS

HTML

 
<form action="" name="form1">
    <label for="male" id="radio1">Your gender?
        <input type="radio" name="gender" id="male" value="male">Male
        <input type="radio" name="gender" id="female" value="female">Female
    </label><br><br>
</form>

CSS
Атрибути
  • type: Визначає тип елемента введення. У випадку радіокнопок, значенням є "radio", що вказує на те, що це радіокнопка.
  • name: Ім'я групи радіокнопок. Всі радіокнопки з однаковим значенням цього атрибута утворюють одну групу. Це дозволяє обирати тільки один варіант з цієї групи.
  • id: Унікальний ідентифікатор радіокнопки. Цей ідентифікатор може використовуватися для зв'язування з міткою (тегом <label>), що полегшує вибір радіокнопки натисканням на текст мітки.
  • value: Значення, яке буде відправлено на сервер, якщо радіокнопка буде вибрана. У цьому випадку, значення "male" для чоловічої статі та "female" для жіночої.

Тег <select>

Тег <select> створює випадаючий список.


HTML
CSS

HTML

 
<form action="" name="form1">
    <label for="">
            <select name="" id="">
                <option value="">1</option>
                <option value="">2</option>
                <option value="" selected>3</option>
                <option value="">4</option>
                <option value="">5</option>
            </select>
        </label>
</form>

CSS

Атрибути

  • name: Ім'я випадаючого списку (не вказане у прикладі).
  • id: Унікальний ідентифікатор випадаючого списку (не вказане у прикладі).

Тег <option>

  • value: Значення, яке передається при виборі опції.
  • selected: Визначає, що дана опція вибрана за замовчуванням.

Тег <textarea>

Тег <textarea> створює текстову область для введення багаторядкового тексту.

HTML
CSS

HTML

 
<form action="" name="form1">
    <label for="">
        <textarea name="" id="" cols="20" rows="5"></textarea>
    </label>
</form>

CSS

Атрибути

  • name: Ім'я текстової області (не вказане у прикладі).
  • id: Унікальний ідентифікатор текстової області (не вказане у прикладі).
  • cols: Ширина текстової області в символах. У цьому прикладі - 20.
  • rows: Висота текстової області в рядках. У цьому прикладі - 5.
