Посилання та кнопки за допомогою css

Використання тега <a>

Тег <a> (якір) зазвичай використовується для створення гіперпосилань, які ведуть на інші веб-сторінки. Однак, він також може використовуватися для інших видів дій, таких як відкриття поштового клієнта, здійснення телефонного дзвінка або виконання JavaScript-коду.

Використання тега <a> для створення посилання, яке виглядає як кнопка

Цей підхід є найпростішим, якщо є посилання, яке потрібно стилізувати під кнопку.

Приклад

Посилання як кнопка
HTML
CSS

HTML


<a href="https://example.com" class="button-link">Посилання як кнопка</a>

CSS


.button-link {
    display: inline-block; /* Робимо елемент блочним для налаштування розмірів */
    padding: 10px 20px; /* Внутрішні відступи */
    background-color: #007BFF; /* Колір фону */
    color: #ffffff; /* Колір тексту */
    text-align: center; /* Вирівнювання тексту по центру */
    text-decoration: none; /* Прибрати підкреслення */
    border-radius: 5px; /* Закруглення кутів */
    transition: background-color 0.3s ease; /* Анімація зміни кольору фону */
  }
  
  .button-link:hover {
    background-color: #0056b3; /* Темніший колір при наведенні */
  }

Створення посилання для відправлення електронної пошти

За допомогою атрибута href і префіксу mailto: ви можете створити посилання, яке відкриває поштовий клієнт користувача і створює новий лист з попередньо заповненою адресою.

Приклад

Send email
HTML
CSS

HTML


<a href="mailto:someone@example.com">Посилання як кнопка</a>

CSS

Створення посилання для здійснення телефонного дзвінка

Використовуючи атрибут href і префікс tel:, ви можете створити посилання, яке ініціює телефонний дзвінок, якщо користувач натискає на нього з мобільного пристрою або клієнта, який підтримує дзвінки.

Приклад

+380 99 123 45 67
HTML
CSS

HTML


<a href="tel:+380991234567">Посилання як кнопка</a>

CSS

Створення посилання для виконання JavaScript-коду

Використовуючи атрибут href і префікс javascript:, ви можете створити посилання, яке виконує JavaScript-код при натисканні.

Execute JavaScript
HTML
CSS

HTML


<a href="javascript:alert('Hello World!');">Посилання як кнопка</a>

CSS

Використання тега <button> для створення кнопки, яка виглядає як посилання

Цей метод корисний, якщо потрібно використовувати функціональність кнопки, але стилізувати її під посилання.

HTML
CSS

HTML


<button class="button-link1" onclick="location.href='https://example.com'">Посилання як кнопка</button>

CSS


.button-link1 {
    display: inline-block; /* Робимо елемент блочним для налаштування розмірів */
    padding: 10px 20px; /* Внутрішні відступи */
    background-color: #007BFF; /* Колір фону */
    color: #ffffff; /* Колір тексту */
    text-align: center; /* Вирівнювання тексту по центру */
    text-decoration: none; /* Прибрати підкреслення */
    border-radius: 5px; /* Закруглення кутів */
    transition: background-color 0.3s ease; /* Анімація зміни кольору фону */
  }
  
  .button-link1:hover {
    background-color: #0056b3; /* Темніший колір при наведенні */
  }
