Адаптивний дизайн

Медіа-запити (Media Queries)

Медіа-запити застосовуються для зміни вигляду сайту на різних розмірах екрану. Наприклад, при розширенні екрану менше 768px, навігаційне меню (nav ul) переходить у вертикальний режим, а контент (content) відображається у стовпцевому режимі.

Гнучкі сітки (Flexbox)

Для навігаційного меню (nav ul) та контенту (content) використовується CSS Flexbox. Це дозволяє елементам гнучко адаптуватися до різних розмірів екрану.

Респонсивні стилі

У CSS використовуються відносні одиниці виміру, такі як відсотки та em, для встановлення розмірів елементів. Це забезпечує їх адаптивність до різних розмірів екрану.

HTML


    <!DOCTYPE html>
    <html lang="uk">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Адаптивний сайт</title>
    </head>
    <body>
        <header>
            <nav>
                <ul>
                    <li><a href="#">Головна</a></li>
                    <li><a href="#">Про нас</a></li>
                    <li><a href="#">Послуги</a></li>
                    <li><a href="#">Контакти</a></li>
                </ul>
            </nav>
        </header>
        <main>
            <section class="hero">
                <h1>Ласкаво просимо на наш сайт!</h1>
                <p>Це приклад адаптивного сайту.</p>
            </section>
            <section class="content">
                <article>
                    <h2>Наша місія</h2>
                    <p>Ми прагнемо надавати найкращі послуги.</p>
                </article>
                <article>
                    <h2>Наші послуги</h2>
                    <p>Ми пропонуємо різноманітні послуги для наших клієнтів.</p>
                </article>
            </section>
        </main>
        <footer>
        </footer>
    </body>
    </html>

CSS


    /* Reset CSS */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

header {
    background: #333;
    color: #fff;
    padding: 1em 0;
}

nav ul {
    display: flex;
    justify-content: center;
    list-style: none;
}

nav ul li {
    margin: 0 1em;
}

nav ul li a {
    color: #fff;
    text-decoration: none;
}

.hero {
    background: #f4f4f4;
    padding: 2em 1em;
    text-align: center;
}

.content {
    display: flex;
    flex-wrap: wrap;
    justify-content: center;
    padding: 2em 1em;
}

.content article {
    flex: 1 1 300px;
    margin: 1em;
    padding: 1em;
    background: #fff;
    border: 1px solid #ccc;
    border-radius: 5px;
}

/* Media Queries */
@media (max-width: 768px) {
    nav ul {
        flex-direction: column;
        text-align: center;
    }

    .content {
        flex-direction: column;
    }
}
