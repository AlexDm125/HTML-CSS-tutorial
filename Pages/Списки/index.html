Атрибут start

Атрибут start використовується у впорядкованих списках <ol> для визначення початкового номера або букви першого елемента списку.

list-style-type: upper-roman; start="100"

Кастомні списки з використанням @counter-style

CSS дозволяє створювати власні стилі лічильників за допомогою правила @counter-style. Це дає можливість використовувати нестандартні символи або зображення для маркерів у впорядкованих списках.

list-style-type: blacknwhite;

  • Java
  • Python
  • JavaScript
  • C#
  • C++
  • php
  • TypeScript
  • Ruby
  • Kotlin
  • Swift
