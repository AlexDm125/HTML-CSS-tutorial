Робота з зображеннями

Додавання зображення через background-image:css

background-image: url(шлях_до_зображення.jpg);

  • background-image - встановлює зображення як фон елементу.
  • url(шлях_до_зображення.jpg) - вказує шлях до зображення, яке буде використане як фон.

Додавання зображення за допомогою тегу <img>

<img src="url_зображення.jpg" alt="Опис зображення">

  • src - вказує шлях до зображення.
  • alt - надає альтернативний текст, який відображається, якщо зображення не може бути завантажено або якщо користувач використовує програму читання екрана.

про css в зображенні два

  • width: 500px; - Задає фіксовану ширину зображення на 500px.
  • border-radius: 50%; - Заокруглює кути зображення, роблячи його заокругленим.
  • box-shadow: 0 0 50px lime; - Додає тінь зображенню. Тут зображено тінь із зеленим кольором (lime) радіусом 50px.

CSS border-image

  • border - встановлює товщину, стиль і колір межі.
  • border-image - вказує шлях до зображення, яке буде використане для створення межі, і ширина області зображення, що використовується для межі.

CSS border-image-slice

border-image-slice: 25% - встановлює внутрішню область зображення, яка використовується для межі, як 25% від всіх сторін зображення.
